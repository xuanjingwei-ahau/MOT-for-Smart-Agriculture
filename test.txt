--------------------------------------添加
# 1. 查看有哪些变更
git status

# 2. 添加所有新文件到暂存区
git add .

# 3. 提交到本地仓库，并写下清晰说明
git commit -m “新增模型训练脚本”

# 4. 推送到GitHub（如果是该分支的第一次推送）
git push -u origin main

# 此后，在该分支推送只需：
git push origin main




--------------------------------------删除单个文件


git rm 要删除的文件名.py    # 删除单个文件
git rm -r 要删除的文件夹名/  # 递归删除文件夹

git commit -m “删除冗余的XXX文件”
git push origin main


--------------------------------------整体覆盖


（强烈建议）备份当前远程仓库：在GitHub页面上，点击 Code → Download ZIP 下载完整快照。

确保本地仓库是你想要的状态：所有文件已就位，无用的文件已清理或加入 .gitignore。

执行强制推送：
git push -f origin main
